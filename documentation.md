## How EventChain works

- For Users.

1. Open the link to [EventChain](https://eventchain-ajq.caffeine.xyz/).
2. You will be able to view open events.
3. You can search, filter, sort and register for any event.
4. Some event are fresh while other are paid.
5. For paid event you need to pay with ICP token.
6. Enter your name and email address.
7. You are done.
8. You can click on the event map location for directions.

- For Event Owners

1. Register with ID - Internet Identity
2. Enter your name
3. Create event and add important information - event description, location (log & lat), date, time.
4. Your event will be open for anyone to register on the home page.
